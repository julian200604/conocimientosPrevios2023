Conocimientos de Git

COMANDOS FUNCIONALIDAD
GIT INIT Este comando sirve para inicializar un proyecto con git.
GIT ADD Permite agregar todos los archivos modificados y nuevos al area de preparación para ser incluidos en el siguiente commit.
GIT COMMIT -M "mensaje" Crea un nuevo commit con los cambios en el area de preparación y lo guarda en el historial de versiones del repositorio
GIT CONFIG --GLOBAL USER.NAME "nombre" Se utiliza para configurar el nombre del usuario de forma global.
GIT BRANCH Sirve para ver todas las ramas en el repositorio y resalta la rama actual
GIT CONFIG --GLOBAL USER.EMAIL "correo" Se utiliza para configurar el email de forma global
GIT PUSH Envia tus commits locales al repositorio correspondiente.
GIT LOG Muestra el historial de commits realizados en el repositorio, incluyendo detalles como el autor, la fecha y el mensaje del commit.
GIT CHECKOUT MAIN Cuando por fuera de la rama main me premite ingresar a ella.
GIT REMOTE ADD ORIGIN "enlace del repositorio" Comando para conectar con github
