Conocimientos de JavaScipt

variables

Las variables se utilizan para almacenar datos en la memoria del ordenador var, let, const

Tipos de datos

string Cadenas de texto
Number numeros
Boolean Valores booleanos (true o false).
Array Colecciones de elementos.
Object Objetos con propiedades y valores.

Operadores

+ Suma
- Resta
* multipliacion
/ División
== Igualdad
!= desigualdad
> Mayor que
< Menor que

Operadores logicos

&& And Sirve para hacer varias comparaciones
|| or Su función principal es evaluar dos expresiones booleanas y devolver true si al menos una de las expresiones es verdadera, y false si ambas son falsas.
! not Invierte los valores

Funciones

las funciones son bloques de código que se pueden reutilizar. Se definen con la palabra clave function y pueden recibir argumentos. Ejemplo:
function saludar(nombre) {
console.log(`Hola, ${nombre}`);
}
saludar("Juan"); // Hola, Juan

Estructuras de control

if Permite ejecutar un bloque de código si una condición es verdadera.
else if Utilizado después de un if para especificar una nueva condición si la primera no se cumple.
else Se ejecuta si ninguna de las condiciones anteriores es verdadera.
for Crea un bucle que consiste en tres partes: inicialización, condición de continuación y expresión de iteración.
while Crea un bucle que ejecuta un bloque de código siempre que una condición especificada sea verdadera.
do-while Similar a while, pero siempre ejecuta el bloque de código al menos una vez, incluso si la condición es falsa.
switch Evalúa una expresión y ejecuta el bloque de código asociado al primer caso que coincide con el valor de la expresión, o ejecuta un bloque de código predeterminado si no hay coincidencias.

arrays

os arrays en JavaScript son colecciones ordenadas de elementos! Suelen usarse para agrupar elementos similares o relacionados para poder acceder a ellos y manipularlos de forma conjunta Sintaxis: const nombreArray = [elemento1, elemento2, ..., elementoN];

Objetos

los objetos son entidades fundamentales que permiten almacenar y organizar datos de manera estructurada. Son colecciones de pares clave-valor, donde cada clave es única y asociada a un valor. Estos valores pueden ser cualquier tipo de dato: números, cadenas de texto, booleanos, funciones, otros objetos, etc. Sintaxis:
let persona = {
nombre: "Juan",
edad: 30,
casado: false,
saludar: function() {
console.log("¡Hola!");
}
};
